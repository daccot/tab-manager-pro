# Privacy Policy for Tab Manager Pro

Last updated: 2026-04-21

## Overview

Tab Manager Pro is designed to help users organize and manage browser tabs locally on their device.

We respect your privacy. This extension does **not collect, store, or transmit personal data to external servers**.

---

## Data Handling

### What data is used

The extension accesses and uses the following data **locally within your browser only**:

- Tab information (such as title, URL, and tab state)
- Tab group information
- User preferences and settings
- Saved sessions and local workspace data

---

### How data is used

This data is used solely to:

- Organize and group tabs
- Detect duplicate tabs
- Enable Focus Mode (temporary tab filtering)
- Save and restore user sessions
- Provide a structured tab management experience

---

### Where data is stored

All data is stored **locally in your browser** using Chrome storage APIs.

- No data is transmitted to external servers
- No data is shared with third parties
- No cloud synchronization is performed

---

## Data Collection

Tab Manager Pro does **not**:

- Collect personal information
- Track user activity outside of the extension’s functionality
- Send browsing data to any external service

---

## Permissions Usage

The extension uses only the permissions required for its functionality:

- `tabs`: To read tab information for organization and display
- `tabGroups`: To create and manage tab groups
- `storage`: To save user preferences and session data locally
- `contextMenus` (if enabled): To provide quick actions via right-click menu

These permissions are used strictly for tab management features.

---

## Third-Party Services

This extension does **not use any third-party analytics, tracking tools, or external APIs**.

---

## Remote Code

Tab Manager Pro does not execute remote code.

All scripts are bundled within the extension package.
